\maketitle

Complete the following tasks adding some comments, when appropriate, on
the obtained results.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  Load the \texttt{HappinessAlcoholConsumption.csv} data set.
\item
  Look at the structure of the data set.
\item
  How may observations' How many variables?
\item
  Convert the character variables into factors.
\item
  Create another data set containing the variables \texttt{Country} and
  \texttt{HappinessScore}.
\item
  Compute the mean of the \texttt{HappinessScore} score for each region.
\item
  How many countries have a mean below the global mean?
\item
  Create a new variable as the sum of \texttt{BeerPerCapita},
  \texttt{SpiritPerCapita} and \texttt{WinePerCapita}
\item
  Compute the median of this new variable considering only the north
  hemisphere.
\end{enumerate}
